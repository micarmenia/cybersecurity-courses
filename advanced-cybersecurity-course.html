About This Course

This comprehensive program takes you from foundational concepts to advanced cybersecurity techniques. You'll spend 80 hours building real-world skills that employers value—learning how to both attack and defend systems, work with professional security tools, and think like a security professional.

The course is designed around hands-on practice. Every concept you learn, you'll immediately apply in practical labs. You'll work with Linux, crack passwords, exploit vulnerabilities, configure security tools, and hunt for threats—just like you would in a real cybersecurity job.

Who Should Take This Course

This course is ideal for:

  • Graduates of our Introduction to Cybersecurity course
  • IT professionals wanting to transition into cybersecurity
  • Computer science students seeking practical security skills
  • System administrators looking to understand security better
  • Anyone serious about starting a career in cybersecurity

Prerequisites

You should have basic computer knowledge and ideally some programming experience. While we'll teach you Linux from scratch, having a general understanding of how computers and networks work will help you get more from the course. If you're not sure whether you're ready, consider taking our Introduction to Cybersecurity course first.

What You'll Learn

This course covers six major areas, each building on the last:

1
Foundations
4 hours
  • Understanding the cybersecurity landscape and career opportunities
  • Learning about common cyber threats in real-world scenarios
  • Exploring legal and ethical considerations in security work
2
Operating Systems (Linux)
8 hours
  • Installing and running Linux with virtual machines
  • Mastering the terminal and essential commands
  • Understanding the Linux file system and permissions
  • Working with package managers and installing software
3
Cryptography
12 hours
  • Understanding encoding and hash functions
  • Learning to crack password hashes
  • Working with symmetric encryption (AES)
  • Understanding asymmetric cryptography and digital signatures
  • Working with certificates and TLS
  • Identifying and exploiting weak encryption
4
Networking Essentials
12 hours
  • Understanding OSI and TCP/IP models
  • Working with ARP, MAC addresses, and Layer 2 security
  • Learning IP addressing, subnets, and routing
  • Understanding DNS and how it works
  • Working with network devices (switches, routers, firewalls)
  • Setting up a cloud VPS and deploying a website
5
Offensive Security
22 hours
  • Gathering intelligence with OSINT techniques
  • Scanning networks and discovering vulnerabilities
  • Exploiting web applications and content management systems
  • Performing system exploitation with Metasploit
  • Understanding and bypassing antivirus software
  • Conducting man-in-the-middle attacks
  • Exploiting file upload, XSS, and SQL injection vulnerabilities
  • Practicing skills in CTF competitions
6
Defensive Security
22 hours
  • Understanding defense-in-depth strategies
  • Implementing network perimeter security (firewalls, VPNs, DMZ)
  • Hardening Windows and Linux systems
  • Working with modern defense systems (SIEM, EDR/XDR)
  • Installing and configuring Wazuh for security monitoring
  • Creating custom detection rules and threat hunting
  • Analyzing threats with Splunk
  • Understanding open source and proprietary security tools

What You Can Do After

After completing this course, you'll have the practical skills to:

  • Apply for entry-level cybersecurity positions (SOC Analyst, Security Analyst)
  • Understand both offensive and defensive security perspectives
  • Work confidently with Linux in professional environments
  • Perform security assessments and vulnerability testing
  • Set up and manage security monitoring tools
  • Analyze security events and hunt for threats
  • Harden systems and implement security best practices
  • Participate in advanced CTF competitions
  • Continue learning through specialized security certifications

Learning Format

Each 2-hour session combines theory with hands-on labs. You'll start by understanding a concept, then immediately apply it in a practical exercise. The labs use real tools and scenarios you'd encounter in actual security work.

By the end of the course, you'll have built up a portfolio of practical skills and completed numerous labs that demonstrate your abilities to potential employers.
